Important FAQ

What is the Document Object Model (DOM)?

DOM is a programming interface provided by browser to interact with and manipulate HTML documents. It treats the entire HTML document as a tree-like structure, where each element, attribute, and pice of content is a node in the tree. The root of the tree is the document object, and children of the root are HTML element and text. We can access the DOM in multiple ways in JavaScript like using ID, Class or Tagname etc. We also can manipulate the DOM like create, update or remove element etc. DOM also support Event Handling.

How do you select an element from the DOM?

We can select an element from DOM using several methods. For Example, we can use getElementById() method to select a single element based on its id attribute, By using getElementsByClassName() method we can select multiple element from the DOM. There are some more method for select elements from the DOM, like getElementsByTagName(), querySelector(), querySelectorAll(), getElementsByName() etc.

What is event delegation in the context of the DOM, and why it is useful?

Event delegation is a technique in JavaScript to handle events at the parent level in the DOM rather than individual element. Instead of binding an event listener to each child element, we can attach a single event listener to a parent element, which will handle events for all it's child elements. It so useful for many reason. For example, It improve the performance as it use only one event listener in the parent only rather each child. It also dynamic, means after adding a new child the event listener will also work for this child element. It make the code more cleaner and easy to manageable.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate or access properties directly for common attributes like value, href, src, etc we can Use setAttribute() and getAttribute() methods and To manipulate the style of an element we can use style property or also can modify classList to add or remove classes from an element.
